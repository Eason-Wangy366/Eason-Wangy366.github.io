    • DROP and TRUNCATE TABLE

      • DROP

        DROP will drop (delete) the whole structure in one go i.e. it removes the named elements of the schema. By using this command the existence of the whole table is finished or say lost.

          DROP TABLE IF EXISTS tblCars

      • TRUNCATE

        TRUNCATE use to delete all the rows of a relation (table) in one go. With the help of “TRUNCATE” command we can’t delete the single row.

          TRUNCATE TABLE tblCars;
