How to Upload my Blog to Github

Upload local server

 hugo server -t m10c --buildDrafts

Use http://localhost:1313/ to check.

Upload to my Github

hugo --theme=m10c --baseUrl="https://Eason366.github.io/" --buildDrafts
#Create web page
cd public
#Enter the public folder
git add .
git commit -m #"commit"
git push -u origin master

Use https://eason366.github.io/ to check.

Creat a new Blog

hugo new post/#BlogName.md
